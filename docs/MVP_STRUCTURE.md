# MVP struktura Call CRM

## Cíl MVP

- evidence firem
- historie hovorů
- kdo volal
- follow-up (kdy volat znovu)
- předání leadu
- schůzka / obchod (základně)

## Připravené části

- routy a controllery pro přehledy modulů
- modely a migrace entit
- dashboard s metrikami
- Blade layout a placeholder stránky

## Doporučený další postup

- dokončit Breeze auth scaffolding (`blade`)
- CRUD formuláře pro firmy a hovory
- CRUD pro follow-upy / lead transfery / schůzky
- seed data a feature testy
- později import z XLSX
